### Supported frameworks

- .NET 3.5+
- .NET 4.5.1+
- .NET Core 1.0+
- .NET Core 2.0+

{{% notice tip %}} In your .csproj add <LangVersion>latest</LangVersion> to allow the async Task Main method. {{% /notice %}}

### Supported databases

- PostgreSQL
- SQLite
- SQL Server
- MySQL / MariaDB
